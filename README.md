# BBeutify
A plugin using js-beautify to de-minimize JS and CSS in BBEdit

## Install

### command line:
`git clone https://github.com/LoneStarMac/BBeautify.git`

`cd BBeautify`

`install.sh`

### manual:

1) Open BBeautify.py in this repo
2) Tap [...] and "Download" or press <kbd>⌘</kbd><kbd>shift</kbd><kbd>s</kbd>
3) Open the download directory
4) Copy BBeautify to `/Application Support/BBEdit/Text Filters` (path varies by install method)
